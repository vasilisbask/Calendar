
# Calendar

This code is part of an exercise for the Object-Oriented Programming II course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a calendar as 
a windowed application.

## Usage

Compile using

```
mvn compile
```

Create a jar using

```
mvn package
```

Run main using

```
java -cp .\target\Mainex-1.0-SNAPSHOT.jar org.example.Main 
```
